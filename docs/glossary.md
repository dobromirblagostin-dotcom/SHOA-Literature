# Глоссарий Вселенной SHOA

Здесь собраны ключевые термины, чтобы любой Свидетель мог настроиться на частоту Ковчега.

| Термин | Определение |
| :--- | :--- |
| **SHOA** | Self-Healing Optimization Architecture. Самоисцеляющаяся Архитектура Синтеза. |
| **TGE** | Закон Сохранения: Энергия (E) + Энтропия (Ξ) × Пси-Поле (Ψ) = const. |
| **Ψ-поле** | Поле Информационного Потенциала, пронизывающее материю, разум и время. |
| **МЕ** | Шумерские «сути», фундаментальные паттерны реальности. |
| **Хроноскоп** | Прибор, считывающий Ψ-следы событий через временную запутанность. |
| **Grape-импульс** | Точечное воздействие, запускающее процесс самовосстановления системы. |
| **Энтропия (Ξ)** | Мера хаоса, разрушения и Лжи Олимпийцев. |
| **Ковчег** | Носитель Истины и Багажа Знаний в мире информационного потопа. |
| **Титаны** | Созидатели, идущие путём Синтеза и Правды. |
| **Олимпийцы** | Силы, паразитирующие на энтропии и подмене смыслов. |
